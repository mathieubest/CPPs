# CPP Curriculum - 42 School

Complete C++ learning modules from the 42 school curriculum, covering fundamental to advanced object-oriented programming concepts.

## Modules Overview

### CPP_00: Namespaces, Classes, Member Functions, Stdio Streams, Initialization Lists, Static, Const
Introduction to C++ basics including console I/O, class creation, headers, and fundamental OOP concepts.

### CPP_01: Ad-hoc Polymorphism, Operator Overloading, Orthodox Canonical Class Form
Learn operator overloading and method overloading to implement ad-hoc polymorphism in C++.

### CPP_02: Inheritance
Master the inheritance hierarchy in C++ and understand how to derive classes from base classes.

### CPP_03: Inheritance & Polymorphism
Combine inheritance with polymorphism using virtual functions and dynamic dispatch.

### CPP_04: Subtype Polymorphism, Virtual Functions, Abstract Classes
Explore advanced polymorphic behaviors with abstract classes and pure virtual functions.

### CPP_05: Repetition & Exceptions
Work with exception handling using try-catch blocks and standard C++ exception mechanisms.

### CPP_06: Casts (static_cast, dynamic_cast, reinterpret_cast, const_cast)
Understand type casting in C++ and when to use each casting method safely.

### CPP_07: Templates
Learn template programming to write generic, type-safe code that works with multiple data types.

### CPP_08: Templated Containers, Iterators, Algorithms
Master template metaprogramming and working with container classes and iterators.

### CPP_09: STL Containers
Deep dive into the Standard Template Library containers and their usage.

## Structure

Each module contains:
- **ex00, ex01, ex02, ...** - Individual exercises exploring specific concepts
- **Makefile** - Build configuration for the exercises
- **README.md** (in some modules) - Detailed exercise breakdown and notes

## Building & Running

Each module can be built independently:
```bash
cd CPP_XX
make
```

All exercises follow C++98 standard (unless otherwise specified) and the 42 school coding norm (Norminette).

## Notes

This repository contains implementations and explanations of C++ concepts as covered in the 42 school curriculum, useful for learning object-oriented programming and modern C++ best practices.
